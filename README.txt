Tarea 3 PSP 1

DESCARGA

Para descargar el proyecto desde el repositorio de GitHub, seguir los siguientes pasos:

- Abrir en una consola o shell del sistema
- Ubicarse en la raiz de la carpeta de proyectos
- Ejecutar el siguiente comando:

git clone https://github.com/adlopez11/psp1.git proyectopsp1

Se creará una carpeta llamada "proyectopsp1" con todo el contenido del repositorio, se puede cambiar el nombre de la carpeta.


EJECUCIÓN

Luego de compilar el proyecto, ubicados en la raiz de la carpeta, la aplicación se puede ejecutar con el comando:

java -cp target/psp1-1.0-SNAPSHOT.jar edu.uniandes.ecos.ui.Main

La aplicacion usa un archivo plano con las dos listas de valores a los que se les calculará la regresión. Colocar en cada línea del archivo el valor de la primera lista (xi), luego un "tab" y posteriormente el valor de la segunda lista (yi).  El nombre del archivo lo solicitará la aplicación.

Colocar el archivo plano en /src/test/resources/
